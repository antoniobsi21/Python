# Python

Projeto destina-se ao estudo da linguagem de programação Python e algumas de suas bibliotecas.

## Opencv-python

**OpenCV** é uma biblioteca gratuitamente distribuída para o desenvolvimento de aplicações de visão computacional. A mesma é distribuída em diversas linguagens, incluindo python, e pode ser facilmente instalada através do comando "pip3 install opencv-python" (diretório: **opencv**).

## Jogo da adivinhação

Trata-se de uma pequena aplicação desenvolvida em sala de aula de um brincadeira para adivinha uma palvra embaralhada de uma lista de palavras (diretório: **jogoAdv**).
